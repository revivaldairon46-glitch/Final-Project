This folder should contain the watch images used by the site.

Place images with these filenames (or update component imports):
- `hero-watch.jpg` (1920x1080 recommended)
- `about-watch.jpg` (1200x800 recommended)
- `feature-battery.jpg` (800x600)
- `feature-health.jpg` (800x600)
- `feature-rugged.jpg` (800x600)


Suggested sources: Unsplash, Pexels. Optimize images to WebP for best performance.

Example command to download (from project root):
```
mkdir -p src/assets/images
curl -Lo src/assets/images/hero-watch.jpg "https://images.unsplash.com/photo-1519744792095-2f2205e87b6f"
curl -Lo src/assets/images/about-watch.jpg "https://images.unsplash.com/photo-1523275335684-37898b6baf30"
curl -Lo src/assets/images/feature-battery.jpg "https://images.unsplash.com/photo-1525975891371-8a6d46c6f45a"
curl -Lo src/assets/images/feature-health.jpg "https://images.unsplash.com/photo-1517433456452-f9633a875f6f"
curl -Lo src/assets/images/feature-rugged.jpg "https://images.unsplash.com/photo-1523275335684-37898b6baf30"
```
